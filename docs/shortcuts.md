# Keyboard shortcuts

## Global

| Keys            | Action                                                                  |
| --------------- | ----------------------------------------------------------------------- |
| `⌘K` / `Ctrl-K` | Open command palette. Type to search calculators, clients, engagements. |
| `?`             | Open contextual help for the current page.                              |
| `g` `c`         | Go to Calculators.                                                      |
| `g` `s`         | Go to Saved calcs.                                                      |
| `g` `q`         | Go to My queue.                                                         |
| `g` `x`         | Go to Exports.                                                          |
| `g` `,`         | Go to Profile (admin: settings).                                        |

## Workbench

| Keys                | Action                                                                        |
| ------------------- | ----------------------------------------------------------------------------- |
| `↑` / `↓`           | Move between rows.                                                            |
| `Tab` / `Shift-Tab` | Move between columns.                                                         |
| `Enter`             | Edit the current cell. `Escape` cancels.                                      |
| `⌘C` / `⌘V`         | Copy / paste a row.                                                           |
| `⌘D`                | Duplicate the current row below.                                              |
| `Delete`            | Remove the current row.                                                       |
| `⌘Z` / `⌘⇧Z`        | Undo / redo (per-tab history stack).                                          |
| `⌘S`                | Save the current calculation.                                                 |
| `⌘E`                | Export PDF.                                                                   |
| `⌘T`                | New tab.                                                                      |
| `⌘W`                | Close tab.                                                                    |
| `+1m` etc.          | In the date column: advance from the row above by 1 month / `+7d`, `+1y` etc. |
| `⌘.`                | Toggle "Show running balance" column.                                         |
| `⌘/`                | Toggle "Show cumulative totals" column.                                       |
| `⌘⇧U`               | Set unknown on the focused cell (solve-for).                                  |
| `⌘⇧R`               | Reset unknowns.                                                               |

## Calculator forms

| Keys     | Action                            |
| -------- | --------------------------------- |
| `⌘Enter` | Submit (compute the calculation). |
| `⌘R`     | Reset the form.                   |

## Tables / lists

| Keys           | Action                                         |
| -------------- | ---------------------------------------------- |
| `j` / `k`      | Move down / up one row.                        |
| `o` or `Enter` | Open the focused row's detail.                 |
| `e`            | Edit the focused row inline (where supported). |
| `a`            | Archive the focused row.                       |
| `/`            | Focus the search input.                        |

## Modal dialogs

| Keys     | Action                                          |
| -------- | ----------------------------------------------- |
| `Escape` | Close the modal.                                |
| `⌘Enter` | Confirm the primary action (where unambiguous). |

(`⌘` = Cmd on macOS, Ctrl on Windows/Linux.)
